PokerII

先看看 PokerII 的键位图吧：

是不是发现能精简的按键全精简了呢，Esc 跟 `~ 都合并成了一个键，而且没有方向键在使用上会有许多不便。

键位思考与改造方案：

键盘的背面有 4 个拨动开关的，他们的功能分别是：

开关 1：Caps 与左 Win 切换
开关 2：右 Ctrl 与 ~ 切换
开关 3：左 Win 与 Fn 切换
开关 4：键盘写保护，键位编程

结合这 4 个开关，有一个绝佳的方案，就是把 CapsLock 换成 Fn，那方向键的问题就完美解决了，顺手也把 Cmd + ` 的问题给解决了，都能用单手操作了。

方案最终实现的目标如下：

改造步骤

  1. 将开关 1 和 3 都拨到 ON 的位置，效果：Caps 变成了 Fn，左 Win 变成了 Caps，右 Fn 还是 Fn
  2. 打开 Mac 的系统偏好设置，在键盘的修饰键修改中，选择 PokerII
  3. 更改修饰键，如下：
    Caps Lock -> Option
    Control -> Control
    Option -> Command
    Command -> Option
  4. 用拔键器把左侧的 Alt 和 Win 键帽互换个位置，这样就更接近 Mac 的标准布局了。

这样，就可以通过 Caps + wasd 来调整方向，Cmd + Caps + Esc 也能在程序里切换窗口了，都只要一只手搞定哦。
